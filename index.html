Pembuatan website menggunakan HTML, CSS, JavaScript, dan PHP.

1. HTML (HyperText Markup Language) Ini struktur atau kerangka website. Bayangkan HTML itu seperti tulang bangunan: menentukan apa yang ada di halaman. Misal: Judul Paragraf Gambar Tombol Formulir

2. CSS (Cascading Style Sheets) CSS itu untuk mengatur tampilan / desain dari HTML. Bisa diibaratkan cat, warna, dan dekorasi rumah.

3. JavaScript (JS) JS itu untuk interaksi dan dinamika website. Bisa diibaratkan alat listrik dan tombol otomatis: tombol bisa diklik, animasi bergerak, form dicek sebelum dikirim.

4. PHP (Hypertext Preprocessor) PHP itu bahasa server-side: diproses di server sebelum muncul di browser. Fungsinya untuk mengelola data: misal login, database, kirim form, atau menampilkan konten dinamis. Bisa diibaratkan dapur di belakang rumah: kita kirim bahan, diolah, baru keluar jadi makanan.
